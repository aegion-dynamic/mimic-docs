MIMIC Documentation

GPIO Manipulation

MIMIC provides a high-level command interface to manipulate GPIO pins on the STM32 hardware.

Pin Naming Convention

Pins are referenced by their Port and Number.

  • A5 -> Port A, Pin 5
  • B12 -> Port B, Pin 12
  • C13 -> Port C, Pin 13 (Built-in LED)

Basic Setup

Set as Output

Configures a pin to drive external components.

PIN_SET_OUT <PIN>
# Example
PIN_SET_OUT A5

Set as Input

Configures a pin to read digital signals.

PIN_SET_IN <PIN> [PULL]
# Example
PIN_SET_IN B0 UP    # Pull-up resistor enabled
PIN_SET_IN B0 DOWN  # Pull-down resistor enabled
PIN_SET_IN B0 NONE  # Floating input

Controlling Outputs

Set High (3.3V)

PIN_HIGH <PIN>
# or short command
PH <PIN>

Set Low (GND)

PIN_LOW <PIN>
# or short command
PL <PIN>

Toggle State

PIN_TOGGLE <PIN>
# or short command
PT <PIN>

Reading Inputs

Read Logic Level

PIN_READ <PIN>
# or short command
PR <PIN>

Response: HIGH or LOW.

Advanced Configuration

Pin Mode

Manually set the hardware mode of the pin.

PIN_MODE <PIN> <IN|OUT|AF|AN>
  • IN: Input
  • OUT: Output
  • AF: Alternate Function (for I2C, SPI, etc.)
  • AN: Analog (for ADC/DAC)

Pin Status

View the complete internal configuration of a pin.

PIN_STATUS <PIN>
# or short command
PS <PIN>

Example Output:

PIN A5:
  Mode:   OUTPUT
  Type:   PUSH_PULL
  Speed:  LOW
  Pull:   NONE
  State:  LOW
